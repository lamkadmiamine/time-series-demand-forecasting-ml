# demand-planning-forecasting

Dans un contexte de forte incertitude, le demand planning vise à estimer la demande client sur un horizon stratégique pouvant aller jusqu’à 18 mois. Cette prévision permet d’anticiper les décisions de supply planning et de production afin de limiter les risques de surstock et de rupture.

Ce projet exploite des méthodes d’IA pour passer d’une prévision ponctuelle à une prévision probabiliste, permettant de générer plusieurs scénarios de demande et d’alimenter la prise de décision en supply planning.
